##Project Name: 'OpenShelf'
##Purpose:
--> OpenShelf is a seamless and modern web application designed to digitize the traditional library experience. Users can explore a vast collection of books, filter by categories, and borrow titles digitally. The platform prioritizes security and performance using BetterAuth, Next.js, and MongoDB.
##live URL : https://b13-a08-open-shelf-library-two7-n5ktw201m-reachme1.vercel.app/
##key features:
--> Extensive Book Collection – Browse a variety of books across different categories with detailed information for each book.
--> Book Discovery & Filtering – Users can search for books by category and apply filters to easily find specific titles within the platform.
--> Authentication System – Supports both email/password login and Google sign-in for a smooth user experience.
